Java NIO (New IO)

Канал (Channels) двунаправленная сущность для чтения и записи
Буфферы (Buffers) оптимизируют процесс чтения или записи
Blocking Non-blocking блокирующий и асинхронный IO что важно для приложений с высокой нагрузкой
Выборочная работа с файлами random access через FileChannel и MappedByteBuffer
Path Files для работы с файлами и директориями
